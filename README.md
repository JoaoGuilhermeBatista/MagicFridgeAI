# MagicFridgeAI

Uma API de gerenciamento de geladeira integrada com inteligência artificial, desenvolvida com Spring Boot.  
*An AI-powered fridge management API built with Spring Boot.*

---

## Português

Projeto que combina gerenciamento de itens de geladeira com chamadas a uma IA externa via WebFlux. Com base nos ingredientes cadastrados, a API é capaz de sugerir receitas e outras informações relevantes.

**Tecnologias**

- Java 21 / Spring Boot 3.5
- Spring Data JPA + Flyway
- Spring WebFlux
- H2 Database
- Lombok

---

## English

A project that combines fridge item management with external AI calls via WebFlux. Based on registered ingredients, the API can suggest recipes and other relevant information.

**Tech Stack**

- Java 21 / Spring Boot 3.5
- Spring Data JPA + Flyway
- Spring WebFlux
- H2 Database
- Lombok
